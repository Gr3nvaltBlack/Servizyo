# Sèvizyo 🏠

A Haitian platform connecting employers and domestic service providers (housekeepers, governesses, yard workers, drivers, babysitters).

Sèvizyo formalizes the informal domestic work market in Haiti by acting as the official employer of service providers—guaranteeing wages in accordance with the Labor Code, a legal contract, and identity verification to reassure employers in a challenging security environment.

## Main features
- Certified profiles with a badge system and skill levels
- Monthly subscriptions and one-off assignments
- Integrated payment via MonCash
- Employer and service provider dashboards
- Automatic WhatsApp notifications
- Comprehensive administration panel
